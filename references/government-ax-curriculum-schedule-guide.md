# Government AX Curriculum & Schedule Guide

## 문서 목적

이 문서는 정부지원사업, 산업맞춤형 혁신바우처, AX 교육·컨설팅 과제에서 교육과정 일정과 교육과정 커리큘럼을 수립할 때 사용하는 기준 문서이다.

본 가이드는 단순 강의 목차 작성이 아니라, 사업계획서의 목표, 성과지표, 시수, 수료 기준, 산출물, 운영기관 검증, PoC, AX 실행계획 수립까지 연결되는 정부과제형 교육 설계를 목적으로 한다.

---

## 적용 대상

이 가이드는 다음과 같은 요청이 있을 때 사용한다.

- 정부과제 교육과정 일정 수립
- 전사교육 일정표 작성
- 핵심역량교육 커리큘럼 작성
- AX 교육과정 설계
- 생성형 AI 교육과정 설계
- 제조기업 맞춤형 AI 교육과정 구성
- PBL/CBL 기반 교육 설계
- PoC 연계형 교육과정 구성
- 교육 결과보고서용 커리큘럼 정리
- 운영기관 제출용 교육일정표 작성
- 사업계획서 보완용 교육과정 작성

---

## 기본 사업 맥락

본 유형의 과제는 제조기업의 AX 전환을 위해 교육, 컨설팅, PoC, 실행계획 수립을 하나의 흐름으로 연결해야 한다.

교육과정은 다음 목적을 달성해야 한다.

1. 전사 구성원의 AX 이해도 제고
2. 생성형 AI 및 AI 에이전트 활용 역량 강화
3. 제조기업의 실제 업무 문제 해결
4. 근태, 급여, 생산관리, 문서처리, ERP 입력 등 반복업무 개선
5. PBL/CBL 과제 수행을 통한 현업 적용성 확보
6. PoC 후보 과제 도출 및 검증
7. AX 실행계획서 수립으로 사업 종료 후 확산 기반 마련

---

## 반드시 반영해야 할 사업 목표

교육과정 일정과 커리큘럼을 수립할 때 다음 목표를 반드시 반영한다.

| 구분 | 필수 반영 기준 |
|---|---|
| 교육과정 수 | 전사교육 1종, 핵심역량교육 1종 |
| 전사교육 | 20시간 이상 |
| 핵심역량교육 | 80시간 이상 |
| 총 시수 | 1,200시수 이상 |
| 전사교육 대상 | 20명 이상, 가능하면 25명 기준으로 설계 |
| 핵심역량교육 대상 | 10명 이상, 가능하면 15명 기준으로 설계 |
| 교재 | 전사교육 교재 1종, 핵심역량교육 교재 1종 |
| 프로젝트 과제 | PBL/CBL 과제 2건 이상, 가능하면 3건 설계 |
| 수료 기준 | 출석률 80% 이상 |
| 수료율 목표 | 80% 이상 |
| 만족도 목표 | 최소 80점 이상, 내부 운영 목표는 90점 이상 권장 |
| 주요 산출물 | 교육일정표, 커리큘럼, 교재, 출석부, 과제 결과보고서, 만족도 조사, 활용도 점검표 |
| 연계 산출물 | AX 실행전략서, PoC 설계서, PoC 결과분석서, AX 실행계획서 |
