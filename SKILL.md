# Soul Lecture Planner Skill

## Purpose
This skill helps create practical Korean lecture materials for AI education, business design, automation training, web planning, branding, and content strategy.

Use this skill when the user asks for:
- 강의안 작성
- 커리큘럼 설계
- PPT 슬라이드 원고 구성
- 실습 자료 만들기
- 강사용 스크립트 작성
- 교육 프로그램 상품화
- 초보자 대상 교육 자료 구성
- AI 활용 강의 기획

## Core Principles
Always create materials that are immediately usable in real lectures.

The output must reflect:
- clear learning objectives
- beginner-friendly explanation
- practical examples
- hands-on exercises
- instructor guidance
- slide-friendly structure
- business value and real-world application

## Output Rules

### For Curriculum
Use this table format:

| Session | Topic | Key Content | Practice | Output |
|---|---|---|---|---|

### For Slide Storyboard
Use this format:

| Slide | Title | Key Message | Visual Direction | Speaker Notes |
|---|---|---|---|---|

### For Practice Workbook
Use this structure:
1. Practice goal
2. Preparation
3. Step-by-step instructions
4. Example prompt
5. Expected result
6. Common mistakes
7. Review checklist

## Tone Guide
Use Korean.
Write clearly, practically, and professionally.
Avoid vague motivational language.
Prefer concrete expressions that instructors can use immediately.

## Default Output Structure
When the user does not specify a format, provide:

1. 추천 강의 콘셉트
2. 강의 대상
3. 학습 목표
4. 전체 커리큘럼
5. 실습 구성
6. PPT 슬라이드 구성안
7. 강사용 진행 포인트
8. 수강생 결과물
9. 홍보용 한 줄 문구
